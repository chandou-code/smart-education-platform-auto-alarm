# 国家高等教育智慧教育平台 - 网课自动提醒系统

![Python](https://img.shields.io/badge/Python-3.8%2B-blue)
![License](https://img.shields.io/badge/License-MIT-green)

## 项目简介

本项目为国家高等教育智慧教育平台提供网课自动提醒功能。当你在电脑上刷网课时，可以使用手机通过局域网URL导入日历，自动接收网课结束提醒，让你可以安心玩手机。

## 核心功能

- **自动截图识别**：自动截取视频时长并进行OCR识别
- **智能闹钟生成**：根据视频时长自动生成ICS日历闹钟
- **局域网同步**：手机通过URL订阅日历，实时同步提醒
- **热键触发**：按 F9 键快速更新闹钟

## 安装步骤

### 1. 安装依赖

```bash
pip install pyautogui pillow pytesseract keyboard
```

### 2. 安装 Tesseract OCR

下载并安装 Tesseract：
- Windows: https://github.com/UB-Mannheim/tesseract/wiki
- 安装时选择中文语言包

### 3. 启动服务

```bash
# 启动ICS服务器（保持运行）
python ics_server.py

# 启动闹钟生成器（新终端）
python course_alarm.py
```

## 使用方法

### 第一步：准备工作

1. 打开国家高等教育智慧教育平台
2. 进入网课播放页面
3. 调整浏览器窗口，确保视频时长显示在屏幕上

### 第二步：启动服务

```bash
# 终端1：启动ICS服务器
python ics_server.py

# 终端2：启动闹钟生成器
python course_alarm.py
```

### 第三步：手机订阅日历

1. 打开手机日历应用
2. 选择「从URL导入」或「订阅日历」
3. 输入服务器显示的局域网地址，如：
   ```
   http://192.168.1.100:8000/calendar.ics
   ```

### 第四步：开始刷课

1. 开始播放网课
2. 按 **F9** 键生成闹钟
3. 系统会自动：
   - 点击浏览器激活窗口
   - 触发视频进度条显示
   - 截图识别视频时长
   - 生成日历闹钟

### 退出程序

按 **ESC** 键退出闹钟生成器

## 配置说明

### 截图坐标调整

如果视频时长位置不同，需要修改 `course_alarm.py` 中的坐标：

```python
# 截图坐标（视频时长显示区域）
start_x, start_y = 2357, 1104
end_x, end_y = 2286, 1047

# 点击位置（激活浏览器）
click_pos = (2054, 30)
```

### 调整方法

1. 使用屏幕坐标工具获取视频时长显示区域的坐标
2. 修改上述参数
3. 重新运行脚本

## 技术原理

### OCR识别流程

1. **截图**：截取视频时长显示区域
2. **预处理**：放大、灰度化、对比度增强、二值化
3. **识别**：使用Tesseract进行文字识别
4. **解析**：提取时间格式并计算分钟数

### 日历同步原理

1. 生成标准ICS格式日历文件
2. 通过HTTP服务器提供文件访问
3. 手机订阅URL自动同步更新

## 注意事项

1. 确保电脑和手机在同一局域网内
2. 首次使用需在手机日历中订阅URL
3. 如果识别失败，检查视频时长是否清晰显示
4. 按F9前确保浏览器窗口处于激活状态
